# CanvasX-IDE
CanvasX-IDE 是一款革命性的桌面端 IDE 工具，它将 Vue 3 + TypeScript 的声明式开发体验与 Canvas 高性能渲染相结合。开发者使用熟悉的 Vue 语法编写组件，最终输出为纯 Canvas 驱动的 Web 应用，性能远超 Flutter Web 和传统 DOM 方案。


✨ 核心特性
⚡ 极致性能 - 纯 Canvas 渲染，无 DOM 开销，60fps 丝滑体验

📦 超小体积 - 运行时仅 28KB (gzipped)，打包应用 < 50KB

🎨 Vue 语法 - 完整支持 Vue 3 Composition API + TypeScript

🖥️ 桌面原生 - 基于 Tauri 构建，打包后仅 8MB，内存占用 < 180MB

🔥 热重载 - 编辑即预览，毫秒级编译反馈

📱 移动优先 - 专为 WAP/H5 页面优化，完美适配移动端

🎯 零依赖输出 - 打包产物无任何外部依赖，单文件部署
